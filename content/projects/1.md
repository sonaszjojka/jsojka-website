# GamiLife

Web-based task and habit management system with gamification elements

## Repositories
- Frontend: https://github.com/sonaszjojka/gamilife-frontend
- Backend: https://github.com/sonaszjojka/gamilife-backend

## Project Description

GamiLife is a web application designed to improve user productivity by combining task management, habit tracking, and budget control. The system leverages gamification mechanisms to increase user motivation and engagement.

The application supports both individual use and group collaboration, offering features such as communication, shared goals, and rankings.

## Key Features

- Task and habit management  
- Pomodoro technique  
- Budget module (expenses, savings goals, reports)  
- Group system (shared tasks, budget, chat)  
- Gamification:
  - experience points (XP)
  - levels and rewards
  - virtual avatar
  - achievements
- Notification system (email + in-app)  
- Shop (individual and group)  
- Monthly budget reports  

## Main Goals

- Increase user productivity  
- Support habit building  
- Motivate users through gamification  
- Integrate task management, budgeting, and communication
