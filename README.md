# Jsojka Website

A modern personal website built with Nuxt and Vue.

## Overview

This is a personal portfolio/website project built with **Nuxt 4**, **Vue 3**, and **TypeScript**. The site is designed to be fast, lightweight, and easy to maintain.

## Tech Stack

- **Frontend Framework**: [Nuxt 4](https://nuxt.com/) & [Vue 3](https://vuejs.org/)
- **Styling**: CSS & Nuxt UI
- **Content Management**: Nuxt Content
- **Type Safety**: TypeScript
- **Utilities**: normalize.css

## Language Composition

- Vue: 93.7%
- CSS: 3.7%
- TypeScript: 2.6%

## Getting Started

### Prerequisites

- Node.js (18+ recommended)

### Installation

```bash
npm install
# or
pnpm install
```
